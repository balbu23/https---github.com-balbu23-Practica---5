¿Qué es HTML?

-Es un lenguaje de marcado que permite hacer la estructura de una página web.

¿Qué es CSS?

-Es un lenguaje que permite dar forma, color y otras caracteristicas visuales a una página web.

¿Qué es JavaScript?

-Lenguaje de programación que permite crear funciones dinámicas en páginas web.
